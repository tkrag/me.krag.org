More roaming for the real world<\/strong> – Swiss Virtual Mobile Operator t-net, announced the WeRoam<\/a> service. The service allows users to seamlessly roam between GSM networks and W-LAN hotspots for wireless data coverage. Using GSM authentication for both GSM and W-LAN service, seems like a clever move. It brings the security of a hardware-based SIM-card to W-LAN authentication, but more importantly it allows users to roam seamlessly, and receive a single bill for this best-of-both-worlds service.
\nAlthough the service initially only covers W-LAN roaming, the goal of GSM\/GPRS and WLAN roaming is clear.
\nCombine this with Nokia’s new D211<\/a> W-LAN and GPRS data card, and suddenly we’re getting somewhere.<\/p>\n
