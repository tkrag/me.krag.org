WiFi Delivers For Developing Countries<\/a><\/p>\n

From austrralia.internet.com, comes this article about MIT Media Lab India’s DakNet<\/a> project.<\/p>\n

A simple store-and-forward WiFi system, using a government bus as a central linkage. The bus contains a simplem WiFi installation and server, and when in range of one of the outlying information kisok’s it sync’s data for later processing.<\/p>\n

This is one of a number of interesting wireless pilot’s around the world, and points at some of the possible solutions for delivering simple, 2-way information access to remote regions. <\/p>\n

DakNet is a patented wireless package that does away with base stations. Instead, a custom Mobile Access Point (MAP) is mounted on, and powered by, a vehicle. A government bus is being used in a current implementation that transports data to and from wireless-enabled computers. The plan is to export DakNet to various parts of the globe.<\/bq><\/p>\n","protected":false},"excerpt":{"rendered":"
